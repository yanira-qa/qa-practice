# Test Case: Login validation message when username and password fields are left empty

**Preconditions:**
- User is on the login page.
- Username and password fields are empty.

**Steps to Reproduce:**
1. Open the login page: [https://www.saucedemo.com](https://www.saucedemo.com)
2. Leave the username field empty.
3. Leave the password field empty.
4. Click the **Login** button.

**Expected Result:**
- The system displays a login validation error message.

**Actual Result:**
- The system displays the correct validation error message.
